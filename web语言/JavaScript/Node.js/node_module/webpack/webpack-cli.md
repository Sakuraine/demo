# webpack-cli

> webpack终端命令行工具

## 命令行配置

### 查看帮助

```shell
# 查看cli可用的所有选项
$ webpack --help
$ webpack -h
```

将webpack打印为json

```shell
$ webpack --json
$ webpack --json > stats.json
```

## 运行

在 `package.json` 文件里添加`"scripts"`对象

```json
{
  "scripts": {
    "start": "", // 配置参数
  },
}
```

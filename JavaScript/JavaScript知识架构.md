##### instanceof 操作符

```js
if (value instanceof Array){
	// 对数组执行某些操作
}
```

##### Array.isArray()

```js
if (Array.isArray(value)){
  // 对数组执行某些操作
}
```

支持 Array.isArray()方法的浏览器有 IE9+、Firefox 4+、Safari 5+、Opera 10.5+和 Chrome

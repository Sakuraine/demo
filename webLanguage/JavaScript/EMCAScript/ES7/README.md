### Array.prototype.includes(value, fromIndex)

> 用来判断数组里是否包含一个指定的值

> 语法

arr.includes(value, index);

> 参数

value

用来判断数组里是否包含一个指定的值

index

开始搜索的索引

> 返回值

Boolean

> Example

```js
var array = [1, 2, NaN];
console.log(array.includes(2));
// true
console.log(array.includes(NaN));
// true
```
